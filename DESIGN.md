# デザインガイド — どこか奇妙な職業体験「真夜中の学芸員」LP

## コンセプト

閉館後の美術館で、トニー・アウスラーの投影作品に囲まれながら学芸員として職業体験をするイマーシブイベント。

LPのデザインは **「暗闇の中に光が投影される」** というアウスラー作品の本質をそのままWebに持ち込んでいる。
闇（黒い背景）の上にピンク/紫の光（アクセントカラー）が浮かび上がり、異常な出来事だけがさらに強く光る。

## カラーシステム

### 3つのレイヤー

```
┌─────────────────────────────────────────────────┐
│  Layer 1: 闇 — 背景・パネル                      │
│  #020105 / #09060f / rgba(14,10,20,0.88)         │
│  閉館後の美術館の暗さ。ほぼ真っ黒。                 │
├─────────────────────────────────────────────────┤
│  Layer 2: 投影光 — ベースアクセント                │
│  #ff1f75 (ピンク) / #7a45ff (紫) / #a8edff (青)  │
│  全セクションに広く使用。世界観そのもの。            │
├─────────────────────────────────────────────────┤
│  Layer 3: 異常 — Anomalyスポットライト             │
│  #ff5c9a + text-shadow多重                       │
│  通常のピンクより明らかに明るい強発光。              │
│  スクロール連動で1箇所ずつ光る。                    │
└─────────────────────────────────────────────────┘
```

### カラーパレット

| 変数名 | 値 | 用途 |
|--------|-----|------|
| `--accent` | `#ff1f75` | ピンク。ベースカラー。CTA・ラベル・ボーダー・区切り線 |
| `--accent-dark` | `#8d123e` | 暗ピンク。チェックボックス |
| `--violet` | `#7a45ff` | 紫。CTAグラデーション・グリッチ演出 |
| `--ice` | `#a8edff` | アイスブルー。情報補助色（時間・リンク・受付） |
| `--text` | `#fffaf7` | メインテキスト。温かみのある白 |
| `--muted` | `#b9b1c4` | 補足テキスト。薄いグレー紫 |
| `--dim` | `#756f80` | 控えめなラベル |
| `--line` | `rgba(255,31,117,0.38)` | ピンク透過の区切り線 |
| `--paper` | `#f4f0ec` | メールカード背景（明るいUI用） |
| `--ink` | `#1c1b20` | メールカードのテキスト |

### 色の使い分けルール

- **ピンク/紫** → ベースの世界観。どのセクションでも使ってOK。投影アートの光そのもの
- **アイスブルー** → 情報を伝える補助照明。時間、場所、リンク、受付情報など
- **白/グレー** → 本文テキスト、パネル背景、微弱なボーダー
- **ピンクの強発光** → Anomalyシステム専用。通常のピンクより明らかに眩しい

**注意: ピンク/紫を構造要素から外すと全体が灰色になり魅力が消える。このサイトではピンクがベースであり、異常は「色の有無」ではなく「光の強度」で差をつける。**

## Anomaly スポットライトシステム

### 概要

異常を示すキーワードが画面の検知バンド（画面の30%〜70%位置）に来ると、通常のテキストより強い発光で光る。常に1つだけ光り、次が来たら前のは0.5秒かけてフェードアウト。

### 発光の仕様

```css
/* 通常テキスト → 周囲と同じ色 */
color: inherit;

/* アクティブ → 強発光 */
color: #ff5c9a;
text-shadow:
  0 0 6px rgba(255,31,117,0.8),
  0 0 20px rgba(255,31,117,0.5),
  0 0 48px rgba(122,69,255,0.35);

/* クロージングライン → さらに強い */
text-shadow:
  0 0 8px rgba(255,31,117,0.9),
  0 0 28px rgba(255,31,117,0.6),
  0 0 64px rgba(122,69,255,0.4);
```

### 速度

| パラメータ | 値 | 場所 |
|-----------|-----|------|
| 光る速度 | 0.2s | styles.css `.anomaly-text` transition |
| 消える速度 | 0.5s | styles.css `.is-anomaly-fading` transition |
| fadingクラス除去 | 500ms | script.js `ANOMALY_FADE_MS` |
| シーケンス間隔 | 1400ms | script.js `advanceSequence` setTimeout |

### 2つのモード

**単体anomaly** — スクロール連動で1つずつ光る
```html
<span class="anomaly-text" data-anomaly>おかしな出来事</span>
```

**シーケンスanomaly** — 親要素が画面に入ると時間差で順番に光る
```html
<p data-anomaly-sequence>
  <span class="anomaly-text" data-anomaly>最初に光る</span>
  <span class="anomaly-text" data-anomaly>次に光る</span>
</p>
```

## インタラクション

### KV チェックリスト

1. ユーザーが3つのチェックボックスを全てチェック
2. フリッカー演出（0.95秒）
3. 目の切替動画が再生（デスクトップ: `kv-eye-switch.mp4` / モバイル: ランダムで猫動画）
4. 5.3秒後にフリッカーで元に戻る
5. 3つ目のチェックだけ外れて注意パルスが出る（「動き出す展示物がないか」のチェック）

### スクロール Reveal

`.reveal` クラスを付けた要素は初期状態で `opacity: 0; translateY(18px)`。
画面の下12%に入ると `0.75s ease` でフェードイン。一度表示したら解除（再発火しない）。

### ヘッダー

- 初期状態は非表示（`opacity: 0; translateY(-100%)`）
- 18px以上スクロールすると表示
- スクロール後はコンパクト化（68px → 56px）、ボーダーがピンクに

### 固定CTA

- イントロセクションのCTAを通過すると画面下部に固定CTAが表示
- モバイル: 画面幅いっぱい、下部グラデーション背景
- デスクトップ: 右下にフローティング

## prefers-reduced-motion 対応

- 全アニメーション・トランジションを `0.001ms` に
- `.reveal` は初期状態で即表示（`opacity: 1; transform: none`）
- `.cat-walk`（猫動画）は `display: none`
- anomalyのtransitionを無効化
- チェックリストCTAの遷移遅延を0msに
